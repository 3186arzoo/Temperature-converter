# Temperature Converter
A simple and responsive temperature converter web application built using HTML, CSS, and JavaScript. This tool allows users to
convert temperatures between Celsius, Fahrenheit, and Kelvin in real time.

# Technologies Used
HTML
CSS 
JavaScript

# Features
Convert between Celsius, Fahrenheit, and Kelvin Real-time conversion as you type Clean and responsive UI Input validation

# How It Works
Enter a temperature value in any one of the input fields. The script automatically calculates and 
fills in the equivalent values in the other two temperature units. Input validation ensures proper numeric entries.
